## Description

ZetPayOffers is a static front-end website for a cryptocurrency and gaming offers aggregator platform, live at [zetpayoffers.eu](https://zetpayoffers.eu/). Built with zero frameworks, no build tools, and no backend — just clean HTML, CSS, and vanilla JavaScript delivered straight to the browser.

The site features a dark-themed design built around a deep violet accent palette, with a layered CSS architecture split across global tokens, typography utilities, shared components, and per-page stylesheets. Interactivity is handled by three focused JavaScript files covering mobile navigation, accordion toggling, and a tab switcher with a smooth sliding indicator animation. Deployed via cPanel, it's intentionally lightweight and fast — no bundler, no runtime, no overhead.
