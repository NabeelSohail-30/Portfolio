# Project Template
# One file per project. Fill every field. Notes in (parentheses) explain options — delete the notes when filling in.

---

## title
(Full project name as you want it displayed on the card)
Example: AI CCTV Surveillance System


## tagline
(One punchy sentence — max ~15 words. This is the card description visible at a glance.)
Example: Real-time suspicious activity detection in CCTV footage using computer vision.


## category
(Short label shown above the title. 1–3 words. Think of it as a genre tag.)
Example: AI / Computer Vision
Options: AI / NLP · AI / Computer Vision · AI / Agents · Full Stack · Backend · Mobile · DevOps · Open Source · Research


## icon
(Font Awesome icon class — just the icon name, no "fas fa-" prefix needed.)
Example: video
Browse icons: https://fontawesome.com/icons
Suggestions: robot · brain · video · shield-alt · calendar-alt · film · database · cloud · code · comments · chart-line · graduation-cap · lock · globe · microchip


## color
(Visual accent color for the card header block. Pick one.)
Options: blue · cyan · purple · green · orange · red
Default: blue


## tech
(Comma-separated list of tech tags shown at the bottom of the card. Keep to 4–6 max.)
Example: Python, YOLO, OpenCV, Django, AWS


## link_github
(GitHub repository URL. Leave blank if private.)
Example: https://github.com/NabeelSohail-30/project-name
Leave blank: —


## description_long
(Optional. 2–4 sentences for future use — modal / case study. Not shown on the card itself.)
Example: This system uses YOLOv8 for real-time object detection...
