## Description

Explore Dominica is an admin dashboard built on Laravel 11 for managing tourism and adventure content for the island of Dominica. It covers the full content lifecycle — menus, attraction listings, detailed entries, hike trails with GPS tracking data, user achievements, ratings, and push notifications — all managed through a clean server-rendered Blade interface.

The platform supports multi-language content fields for Spanish and French alongside English, making it suitable for Dominica's international visitor base. File uploads handle imagery across listings and backgrounds, with GPS and hike location tracking managed through a dedicated model. Push notifications are built in for real-time visitor engagement. Built as a traditional MVC dashboard with no JavaScript framework — just Tailwind, vanilla JS, and Axios for lightweight AJAX interactions.
